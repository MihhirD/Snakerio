## Overview

SnakeRio is a browser-based implementation of the classic Snake game, designed with a modern interface and smooth gameplay experience. Built using HTML, CSS, and JavaScript, the project focuses on real-time interaction, game logic, and responsive controls.

This project demonstrates the ability to build engaging interactive applications directly in the browser.

---

## Core Features

* Classic snake gameplay mechanics
* Real-time movement and controls
* Collision detection system
* Score tracking
* Responsive and smooth UI
* Lightweight and fast performance

---

## Tech Stack

| Technology | Purpose                      |
| ---------- | ---------------------------- |
| HTML5      | Structure                    |
| CSS3       | Styling and layout           |
| JavaScript | Game logic and interactivity |

## How to Play

* Use arrow keys to control the snake
* Eat food to grow in length
* Avoid collisions with walls and yourself
* Try to achieve the highest score possible
